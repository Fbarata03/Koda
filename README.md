# Sonic Koda 🦔

Um jogo de plataforma estilo Sonic, construído com **Electron + Vite + React** e renderizado em **Canvas 2D**.

## Como jogar

| Tecla | Ação |
|---|---|
| `←` `→` | Mover |
| `↑` ou `Space` | Pular |
| `R` | Reiniciar |

- Coleta anéis dourados para ganhar pontos
- Pisa em cima dos inimigos para destruí-los (+100 pts)
- Chega até a estrela giratória no final da fase
- Se levar hit sem anéis, perde uma vida
- 3 vidas no total — boa sorte!

## Como rodar

```bash
npm install
npm run dev
```

> Requer Node.js 18+

## Tech Stack

- **Electron** v32 — janela desktop
- **Vite** + **React** — frontend
- **Canvas 2D** — engine do jogo (sem libs externas)
